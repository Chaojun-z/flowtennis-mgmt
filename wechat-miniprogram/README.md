# FlowTennis 微信小程序

这是 FlowTennis 教练端原生微信小程序项目。

## 使用方式

1. 用微信开发者工具打开 `wechat-miniprogram` 目录。
2. 拿到真实 AppID 后，替换 `project.config.json` 里的 `appid`。
3. `miniprogram/config.js` 里的 `API_BASE_URL` 当前指向 `https://www.flowtennis.cn/api`。
4. 微信后台补齐合法 request 域名、消息模板和隐私保护指引配置。

## 当前占位

- `appid`: `touristappid`
- `API_BASE_URL`: `https://www.flowtennis.cn/api`
